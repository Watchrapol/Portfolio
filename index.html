MyPortfolio

AuyDev Portfolio (HTML/CSS/JS)

พอร์ตนี้ผมเขียนด้วยมือทั้งหน้าเพื่อโชว์สกิลพื้นฐานจริง ๆ: โครงสร้าง HTML semantic + CSS Grid/Flex และ JS ที่พอดีตัว พร้อม micro-interactions ที่ไม่กวนสายตา (IntersectionObserver คุมแอนิเมชันให้ re-run เมื่อเลื่อนเข้า/ออก, Lottie ใน hero, Swiper ที่ตั้ง observer แก้ปัญหาขนาดเพี้ยน) โฟกัส performance/a11y เช่น lazy asset, สีคอนทราสต์, focus ring และโค้ดที่อ่านง่ายต่อยอดได้

This portfolio is hand-coded end to end to showcase real fundamentals: semantic HTML structure, CSS Grid/Flexbox, and just-enough JavaScript. It uses calm micro-interactions—an IntersectionObserver to re-run animations as sections enter/exit, a Lottie animation in the hero, and a Swiper carousel configured with observers to avoid sizing glitches. I focus on performance and a11y: lazy-loading assets, proper color contrast, visible focus rings, and clean, extensible code.
